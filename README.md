# Food Management

# Technology-Used

- MongoDB
- Express.js
- Angular
- Node.js


# How_To_Run

- ## How To run backend 

```
$ git clone https://github.com/jonathankinggold/food-manage-MEAN
$ cd CanteenManagement

$ cd backend
$ npm install

$ npm run dev
```

- ## How To run Front-end App(Angular)

```
$ git clone https://github.com/jonathankinggold/food-manage-MEAN
$ cd CanteenManagement

$ cd frontend
$ cd canteen
$ npm install

$ ng serve 
```
